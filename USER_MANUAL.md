## 🎯 系统简介

### 系统概述

海龙咨询官网是一个专业的企业门户网站，提供以下核心功能：

- 📢 **公告发布**: 工程建设公告、政府采购公告
- 📄 **信息公开**: 政策法规、行业动态、企业信息
- 🏢 **企业展示**: 公司简介、业务范围、资质荣誉
- 📊 **数据统计**: 访问量统计、内容统计
- 🔍 **全局搜索**: 快速查找公告和信息

### 公告管理

#### 工程建设公告

**访问路径**: 侧边栏 → 公告管理 → 工程建设公告

##### 1. 查看公告列表
- 显示所有工程建设公告
- 支持搜索和筛选
- 显示发布状态

##### 2. 添加新公告

**操作步骤**:
```
1. 点击"添加公告"按钮
2. 填写公告信息：
   - 标题（必填）
   - 内容（必填，使用富文本编辑器）
   - 地区（选择省/市/区）
   - 发布日期（必填）
   - 来源（选填）
3. 上传附件（如有）
4. 选择发布状态：
   - 已发布：立即在前台显示
   - 草稿：保存但不显示
5. 点击"保存"按钮
```

**富文本编辑器使用**:
- **加粗**: 选中文字，点击B按钮
- **斜体**: 选中文字，点击I按钮
- **标题**: 选择标题级别（H1-H6）
- **列表**: 点击列表按钮
- **插入图片**: 点击图片按钮，上传图片
- **插入链接**: 选中文字，点击链接按钮
- **对齐**: 选择左对齐、居中、右对齐
- **撤销/重做**: 使用撤销和重做按钮

##### 3. 编辑公告

**操作步骤**:
```
1. 在列表中找到要编辑的公告
2. 点击"编辑"按钮
3. 修改公告信息
4. 点击"保存"按钮
```

##### 4. 删除公告

**操作步骤**:
```
1. 在列表中找到要删除的公告
2. 点击"删除"按钮
3. 确认删除操作
```

**注意**: 删除操作不可恢复，请谨慎操作！

##### 5. 附件管理

**上传附件**:
```
1. 在编辑公告时，找到"附件"部分
2. 点击"上传附件"按钮
3. 选择文件（支持PDF、Word、Excel等）
4. 等待上传完成
5. 可以上传多个附件
```

**删除附件**:
```
1. 在附件列表中找到要删除的附件
2. 点击"删除"按钮
3. 确认删除
```

**支持的文件类型**:
- 📄 文档: PDF, DOC, DOCX, XLS, XLSX
- 🖼️ 图片: JPG, PNG, GIF
- 📦 压缩包: ZIP, RAR

## ❓ 常见问题

### 前台使用问题

#### Q1: 为什么搜索不到公告？
**A**: 可能的原因：
1. 公告未发布（状态为草稿）
2. 搜索关键词不准确
3. 公告已被删除
4. 尝试使用更通用的关键词

#### Q2: 附件无法下载？
**A**: 解决方法：
1. 检查网络连接
2. 尝试使用其他浏览器
3. 清除浏览器缓存
4. 联系管理员检查附件是否存在

#### Q3: 页面显示不正常？
**A**: 解决方法：
1. 刷新页面（Ctrl+F5）
2. 清除浏览器缓存
3. 更新浏览器到最新版本
4. 尝试使用Chrome或Firefox浏览器

### 后台管理问题

#### Q1: 无法登录后台？
**A**: 检查项：
1. 用户名和密码是否正确
2. 账号是否被禁用
3. 是否使用了正确的登录地址
4. 联系管理员重置密码

#### Q2: 上传文件失败？
**A**: 可能原因：
1. 文件大小超过限制（50MB）
2. 文件格式不支持
3. 网络连接不稳定
4. 服务器存储空间不足

#### Q3: 保存内容失败？
**A**: 解决方法：
1. 检查必填项是否填写完整
2. 检查网络连接
3. 尝试重新登录
4. 联系技术支持

#### Q4: 富文本编辑器无法使用？
**A**: 解决方法：
1. 检查浏览器是否支持
2. 禁用浏览器插件
3. 清除浏览器缓存
4. 使用Chrome浏览器

## 💡 操作技巧

### 前台使用技巧

1. **快速搜索**
   - 使用Ctrl+F打开浏览器搜索
   - 在页面内快速查找关键词

2. **收藏常用页面**
   - 将常访问的页面添加到浏览器收藏夹
   - 使用浏览器书签快速访问

3. **打印页面**
   - 使用Ctrl+P打印当前页面
   - 可以保存为PDF格式

### 后台管理技巧

1. **批量操作**
   - 使用复选框选择多个项目
   - 执行批量删除或批量修改状态

2. **快捷键**
   - Ctrl+S: 保存（部分页面支持）
   - Ctrl+Z: 撤销（富文本编辑器）
   - Ctrl+Y: 重做（富文本编辑器）

3. **内容复制**
   - 可以从Word复制内容到富文本编辑器
   - 建议使用"粘贴为纯文本"避免格式问题

4. **图片优化**
   - 上传前压缩图片可加快加载速度
   - 建议图片大小不超过2MB
   - 使用JPG格式可减小文件大小

5. **定期备份**
   - 重要内容编辑前先复制保存
   - 定期导出数据备份
