# 🚀 快速启动指南

## 第一步：安装依赖

打开 PowerShell，进入项目目录并安装依赖：

```powershell
cd C:\Users\Administrator\Desktop\Protral\hailong-admin
npm install
```

⏱️ 预计耗时：2-5 分钟（取决于网络速度）

---

## 第二步：配置后端 API 地址

在项目根目录找到 `.env.development` 文件，确认后端 API 地址：

```env
VITE_API_BASE_URL=http://localhost:5000
```

> 💡 如果后端运行在其他地址，请修改此配置

---

## 第三步：启动开发服务器

```powershell
npm run dev
```

启动成功后，控制台会显示：

```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:3000/
➜  Network: use --host to expose
```

---

## 第四步：访问系统

在浏览器打开：**http://localhost:3000**

## ❓ 常见问题

### 1. 依赖安装失败

**解决方案**：
```powershell
# 清除缓存
npm cache clean --force

# 删除 node_modules
Remove-Item -Recurse -Force node_modules

# 重新安装
npm install
```

### 2. 启动后无法访问后端

**检查清单**：
- ✅ 后端服务是否已启动
- ✅ 后端运行在 http://localhost:5000
- ✅ `.env.development` 配置是否正确
- ✅ 浏览器控制台是否有错误

### 3. 登录失败

**检查清单**：
- ✅ 账号密码是否正确
- ✅ 后端认证接口是否正常
- ✅ 查看浏览器 Network 面板的请求响应

## 📦 生产部署

### 1. 构建

```powershell
npm run build
```

### 2. 部署文件

构建完成后，将 `dist/` 目录下的所有文件上传到 Web 服务器（Nginx、Apache 等）

### 3. Nginx 配置示例

```nginx
server {
    listen 80;
    server_name admin.yourdomain.com;
    
    root /var/www/hailong-admin/dist;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://localhost:5000;
    }
}
```
